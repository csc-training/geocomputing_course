# Puhti webinterface

<p>&rarr; lightweight entry to the supercomputer</p>

## Puhti web interface

* **For easy interacting with Puhti supercomputer**

- View, download and upload files
- **Terminal to Puhti**
- Info: running jobs, disk usage, project status and Puhti general status
- Launch interactive apps and open them directly from the browser:
    - Desktop with apps: **QGIS**, GRASS, SagaGIS etc
    - **Jupyter**
    - TensorBoard
    - **Visual Studio Code**
    - **RStudio**
    - MATLAB

* [Puhti web interface](https://puhti.csc.fi)
* [CSC Docs: webinterface](https://docs.csc.fi/computing/webinterface/)
* LUMI at the moment does not have a web interface, but will also have a similar one.

## Desktop

The Desktop application provides a desktop environment on Puhti within your web browser. From here you have graphical user interfaces of the following software available:

* QGIS
* GRASS
* SAGA GIS
* SNAP
* Zonation

## What the webinterface is for - and what not

<p>&#128077;</p>

* View, download and upload files
* Open a shell on the login node
* Open a persistent shell on a compute node
* View running batch jobs
* View disk quotas and project status
* Launch interactive apps and connect to them directly from the browser
    * Prototyping and testing
    * Visualizations
* Courses

<p>&#128078;</p>

* Running a script on multiple input files -> array job or other HTC workflow
* Running large / long computations -> batch job
* Running large machine learning jobs -> batch job
